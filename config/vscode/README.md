# VS Code Configuration
The JSON configuration file should work without installing any extensions.

# Extensions
1. [Prettier](https://github.com/prettier/prettier-vscode) - Code formatter.
2.  [Live Server](https://github.com/ritwickdey/vscode-live-server) - Live reload for static & dynamic pages.
3.  [Markdown All in One](https://github.com/yzhang-gh/vscode-markdown) - All you need for Markdown.
4.  [Intellicode](https://github.com/MicrosoftDocs/intellicode)  - AI-assisted development.
5.  [Project Manager](https://github.com/alefragnani/vscode-project-manager) - Easy way to access projects.
6.  [Thunder Client](https://github.com/subasraj/flashpost-support) - Rest API Client for testing without leaving VS Code.
